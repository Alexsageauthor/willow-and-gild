WILLOW & GILD — SEATING CHART BUILDER
======================================
Deploy (same as the game):
 1. Create a GitHub repo (e.g. wg-builder), upload ALL files
    in this zip (index.html + the assets folder — keep the
    folder structure!).
 2. In Vercel: Add New -> Project -> import the repo -> Deploy.
 3. Optional: in Vercel project Settings -> Domains, attach
    builder.willowandgild.com (Vercel shows you the DNS record
    to add at your registrar).

Access code: WG-SEAT-1926
 - change it in index.html (ACCESS_CODES near the top) any time
 - give it to buyers via the "Builder Access" page in their
   Etsy download

Buyer flow: enter code -> add tables (round or long, 1-16 seats
each, any mix) -> type names -> live picture preview -> Download
print-ready PDF (A1 or 24x36) with embedded fonts & watercolor
art -> print at any copy shop.
